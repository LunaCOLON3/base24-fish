# base16-fish
[Base16](http://chriskempson.com/projects/base16/) for [fish](https://fishshell.com)

## Installation
1. If you use [fisher](https://github.com/jorgebucaran/fisher),
```
$ fisher add tomyun/base16-fish
```

2. Run your choice of `base16-*` function
```
$ base16-oceanicnext
```
